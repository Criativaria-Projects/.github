# Criativaria

**Tech também é pra você.**

A Criativaria é uma comunidade brasileira de tecnologia. Aqui você é tratado como capaz, no seu ritmo. Por aqui ninguém caminha sozinho.

Se você já sentiu que "não tem o perfil" para estar na tech — você está no lugar certo.

---

## Por onde começar

- **[Discord](https://discord.gg/criativaria)** — o hub principal da comunidade. Fórum, dúvidas, conexões.
- **[Site](https://criativaria.com.br)** — guias, roadmaps, cursos e vagas atualizadas.
- **[YouTube](https://youtube.com/@criativaria)** — conteúdo em vídeo.
- **[Twitch](https://twitch.tv/criativaria)** — lives de programação ao vivo.
